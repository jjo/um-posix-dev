Estructuración del proyecto

  • Fuentes
    • Interface: .h
    • Implementación: .c -> .o
  • Entorno de trabajo
    • Proyecto: Makefile
      • Relación 
        	objectivo(s): fuentes ...
		commandos ...
      • Objetivos típicos:
        • make all: crea todos los "targets"
        • make clean: limpia "targets", objetos
        • make test: prueba la aplicacion
        • make dist: crea .tar
      • Variables comunes:
        • CFLAGS: parámetros para el compilador
        • LDFLAGS: parám. para el linker
        • LDLIBS: libs. para el linker
      • Operación make/vim
    • Un poco más de vim ...

      Uso de :make, ^Z, :sp {arch}

  • Debugger: gdb
    • Intro: señales, core files.
    • Invocación, uso runtime y postmortem (core)
    • Comandos más comunes:
      • run
      • break {funcion|linea}
      • cont, step
      • bt (backtrace)
      • print variable
      • x dir_memoria
    • Ciclo edición, compilación y debugging
